# EduPlatform - Full Stack E-Learning Platform

## Features

- User Authentication & Authorization
- Role-Based Access (Admin/User)
- Course Enrollment System
- Razorpay Payment Integration
- Dynamic Admin Dashboard
- Course & Lesson Management
- Video Streaming using YouTube Embed
- Drag & Drop Lesson Sorting
- Cart & Enrollment Management
- Responsive UI Design

## Technologies Used

- Java
- Spring Boot
- Spring Security
- Hibernate / JPA
- MySQL
- JSP
- Bootstrap
- JavaScript
- Razorpay API

## Project Overview

EduPlatform is a full-stack e-learning web application where users can browse courses, enroll securely, and watch sections online. Admins can manage courses, sections, payments, and users through a dedicated dashboard.
